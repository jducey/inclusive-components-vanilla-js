A true toggle button

Sometimes we use button elements to submit forms. To be fully compliant and reliable these buttons should take the type value of submit.

But these are only one variety of button, covering one use case. In truth, button elements can be used for all sorts of things, and not just in association with forms. We remind ourselve of this by giving them the type value of button.

So how do we make a button a toggle button? It's a case of using WAI-ARIA as a progressive enhancement. WAI-ARIA offers states that are not available in basic HTML, such as the pressed state.

You don't have to use event.preventDefault() on type="button" controls to suppress form submission.
